# tennis-game-scorer

This app is used for getting the scores for a single player tennis game of 1 set.

### Prerequisites

* Node: v10.16.1 or higher
* NPM: v6.4.1 or higher

### Installing and running the app
Install node modules
```
npm i
```

Running the app
```
npm run start
```

### Running tests
```
npm run test
```
